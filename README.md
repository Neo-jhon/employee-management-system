# Employee Management System

A modern Employee Management System built with React that allows administrators to assign tasks and employees to manage and update their work status.

---

## Features

- 🔐 Admin Login
- 👨‍💼 Employee Login
- 📋 Task Assignment
- ✅ Task Status Management
- 📊 Dashboard
- 💾 Local Storage Persistence

---

## Tech Stack

- React.js
- JavaScript
- Vite
- CSS
- Local Storage

---

## Installation

```bash
git clone https://github.com/Neo-jhon/employee-management-system.git

cd employee-management-system

npm install

npm run dev
```
